# Yemeksepeti Mobile Android Bootcamp Final Project.


## About this project

This project is a simple food ordering application. I learned the most used libraries and structures in android applications, in this project

## Used Technologies

* MVVM Architecture
* Retrofit, okHttp
* Dagger Hilt
* Kotlin Coroutine
* ViewModel - LiveData
* Gson
* Lottie Animations
* RecyclerView
* Fragments
* Room

## Project Structure
------
* Splash Screen
* Onboarding Screens
* Login Screen
* Register Screen
* Home Screen
* Add Restaurant Screen
* Restaurant Detail Screen
* Add Food Screen
* Food Detail Screen
* Order Basket Screen
* Profile Screen
* Update User Screen
* Last Orders Screen
